# CalmTask Voice Assistant

A simple voice-based task assistant Android app.

## Features
- Country, language, gender, mood selection
- Tasks with Done / Later / Skip
- Morning greeting (text-to-speech)
- Night review screen
- Voice command input (typed)
- No always-on mic, no cloud AI, no login, no ads

## Build APK via GitHub Actions
1. Push this repo to GitHub
2. Go to **Actions** tab → **Build Android APK** → **Run workflow**
3. Download the APK from **Artifacts** after build completes (~5 min)

## Install
Enable "Install unknown apps" on your Android phone, then tap the APK.
